#Project 5 : House Price Prediction using Linear Regression

## Objective
The objective of this project is to build a machine learning model that predicts house prices based on different housing features.

## Dataset
Housing dataset containing attributes such as area, bedrooms, bathrooms, stories, parking, air conditioning, and furnishing status.

## Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
1. Loaded the dataset
2. Checked the dataset structure and missing values
3. Converted categorical yes/no columns into numeric values
4. Applied one-hot encoding to furnishing status
5. Split the data into training and testing sets
6. Trained a Linear Regression model
7. Evaluated the model using Mean Squared Error and R² Score
8. Visualized predictions and residuals

## Model Used
Linear Regression

## Result
The model achieved an R² score of 0.659, indicating a reasonably good prediction performance.

## Conclusion
This project demonstrates the use of linear regression for solving a real-world regression problem in house price prediction.
