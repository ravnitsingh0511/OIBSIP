# Project 3: Data Cleaning

## Objective
To clean a real-world dataset by handling missing values, duplicates, inconsistent formatting, and outliers.

## Steps Performed
- Filled missing numerical and categorical values
- Removed duplicate records
- Standardized text columns
- Removed invalid price values
- Treated outliers using percentile filtering

## Output
- Cleaned dataset saved as `cleaned_airbnb_data.csv`
