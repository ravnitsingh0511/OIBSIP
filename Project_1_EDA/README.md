# Project 1: Exploratory Data Analysis (EDA)

## Objective
To perform exploratory data analysis on a sales dataset to identify patterns, trends, and key insights.

## Tool Used
- Microsoft Excel

## Steps Performed
- Cleaned the sales dataset by handling missing and inconsistent values
- Sorted and filtered data for analysis
- Used pivot tables and basic formulas for summarization
- Created charts to analyze sales trends

## Outcome
- Identified key sales patterns and trends
- Prepared clean and structured data for further analysis
