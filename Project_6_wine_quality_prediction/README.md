#Project 6 : Wine Quality Prediction

## Objective
The objective of this project is to predict the quality of wine based on its chemical properties using machine learning classification models.

## Dataset
Wine quality dataset containing features such as acidity, chlorides, density, pH, sulphates, alcohol, and quality.

## Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
1. Loaded the dataset
2. Checked dataset structure and missing values
3. Dropped the unnecessary Id column
4. Split the dataset into training and testing sets
5. Trained three classification models:
   - Random Forest Classifier
   - SGD Classifier
   - Support Vector Classifier (SVC)
6. Evaluated the models using accuracy and classification report
7. Visualized the wine quality distribution
8. Analyzed feature importance using Random Forest

## Models Used
- Random Forest Classifier
- SGD Classifier
- Support Vector Classifier (SVC)

## Result
The models were trained successfully to classify wine quality based on chemical features.

## Conclusion
This project demonstrates the use of classification algorithms in predicting wine quality using real-world chemical data.
