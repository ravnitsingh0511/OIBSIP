# Project 8 : Google Play Store Data Analysis

## Objective
The objective of this project is to analyze Google Play Store data to understand app market trends, app categories, ratings, installs, pricing, and user sentiment.

## Datasets
- apps.csv
- user_reviews.csv

## Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps Performed
1. Loaded both datasets
2. Cleaned the apps dataset by fixing installs, price, reviews, and rating columns
3. Removed duplicates and missing values
4. Analyzed app categories, ratings, installs, and pricing
5. Analyzed user review sentiment distribution
6. Merged app and review datasets
7. Created visualizations for category trends, ratings, installs, and sentiment

## Result
The project successfully identified important insights about app market structure, user preferences, and sentiment trends in the Google Play Store ecosystem.

## Conclusion
This project demonstrates practical data analysis and visualization techniques using real-world app market and user review datasets.
