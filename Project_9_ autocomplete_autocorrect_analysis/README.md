# Project 9 : Autocomplete and Autocorrect Data Analytics

## Objective
The objective of this project is to analyze word data and implement simple autocomplete and autocorrect functionalities using text analytics techniques.

## Dataset
google-10000-english.csv containing common English words.

## Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- difflib

## Steps Performed
1. Loaded the word dataset
2. Cleaned and prepared the text data
3. Analyzed word lengths and starting letters
4. Built an autocomplete function using prefix matching
5. Built an autocorrect function using close word matching
6. Tested both functions on sample inputs
7. Visualized word pattern distributions

## Result
The project successfully generated word suggestions for both autocomplete and autocorrect tasks using a simple English word dataset.

## Conclusion
This project demonstrates the use of text analytics and basic NLP methods for improving text prediction and spelling correction systems.
